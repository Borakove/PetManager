# PetManager

Sistema de gerenciamento para pet shops. Desenvolvido em Java com Swing.

## Funcionalidades

- Tela de Cadastro de clientes e pets
- Tela de Agenda
- Tela de Serviços
- Tela de Configurações
- Tela de Login (primaria)
- Navegação por menu
- Separação por classes

## Tecnologias

- Java SE
- Swing
- NetBeans
- Git
